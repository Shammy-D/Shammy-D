What i'm currently working on

...

I'd like to say my network topology is coming along nicely, but many would find my use of a virtual router uneasy, though it's been working perfectly for over a year now with over 900mbps down and 3 ping!

I setup this website via an Nginx server that i'm running on my second Proxmox cluster, with my SSL certs being handled by ACME certs and proxied by HAProxy via my pfsense vm to Cloudflare, so my network isn't exposed to the internet (as long as Cloudflare is still operational ahah). I also recently setup my own email service, so i can receive emails to the @shamya.net domain, so feel free to email contact@shamya.net if you want to help test out my email capacity! My next project will likely be setting up TrueNas (i need a good NAS) or Nextcloud as a private cloud storage alternative.
